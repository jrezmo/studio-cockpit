Version 2025.10

Compatibility

PTSL SDK 2025.10 uses:

  • Old format:
    • PTSL_VERSION 2025 of the client interface
  • New format:
    • Major Version 2025
    • Minor Version 10
    • Revision Version 0

Pro Tools 2025.10 is compatible with clients using Major Version 1 and higher.
