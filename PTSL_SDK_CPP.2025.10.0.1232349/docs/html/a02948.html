TrackControlId

Description

Structure that identifies a track's automatable control within a track.

This type identifies which section the control is in, what type of control is specified,
optional pan control details, and optional plugin parameter control details.

Control type section control_type
track volume TSId_MainOut TCType_Volume
track lfe TSId_MainOut TCType_Lfe
track mute TSId_MainOut TCType_Mute
track pan TSId_MainOut TCType_Pan; further settings reqired in the pan member
send volume TSId_SendA - J TCType_Volume
send mute TSId_SendA - J TCType_Mute
send pan TSId_SendA - J TCType_Pan; further settings required in the pan member
insert plugin TSId_InsertA - J TCType_PluginParameter; further settings in the required plugin member
