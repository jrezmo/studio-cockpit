Version 2023.03

Compatibility

PTSL SDK 2023.03 uses PTSL_VERSION 1 of the client interface

Pro Tools 2023.3 is compatible with clients using PTSL_VERSION 1 and higher

Changelog

Developer certificates have been deprecated

You no longer need to obtain a developer certificate to establish a PTSL connection to Pro Tools. Use the new RegisterConnection command to establish connections to Pro Tools 2023.3 and higher.

AuthorizeConnection has been deprecated. This command, which was used with certificates in previous versions of the scripting SDK, will continue to work but will be removed in a future release.

Command changes

  • RenameTrack trackId is now deprecated. Use currentTrackName instead.
  • ExportMix filesList is now deprecated. Use mixSourceList instead.
