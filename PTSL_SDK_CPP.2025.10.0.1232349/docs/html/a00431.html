Description

C++ PTSL client test application.

This application is used internally in Avid to test the PTSL Scripting SDK. You can use the application at your own risk to send Pro Tools commands without creating your own program. However, we can't guarantee that it won't be changed or removed in the future.

Please see usage information by running:

ptslcmd -help
