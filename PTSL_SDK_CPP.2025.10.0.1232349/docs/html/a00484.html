Version 2023.12

Compatibility

PTSL SDK 2023.12 uses PTSL_VERSION 2 of the client interface

Pro Tools 2023.12 is compatible with clients using PTSL_VERSION 1 and higher

DEVELOPER CERTIFICATES ARE NO LONGER SUPPORTED

You no longer need to obtain a developer certificate to establish a PTSL connection to Pro Tools. Use the new RegisterConnection command to establish connections to Pro Tools 2023.3 and higher.

New enumerations:

New messages:

  • SetTrackOnlineStateRequestBody

New error response type.

The new ResponseError message replaces previous message in the response_error_json. This will allow processing multiple errors/warnings via protobuf.

Removed messages:

  • VideoData
  • MidiData
  • ClipGroups

Existing command changes:

  • Changes in PasteSpecialOptions enumeration
    • Merge is deprecated. Use MergeMidi instead
  • Changes in MemoryLocationReference enumeration
    • MLR_FollowTrackTimebase value was added
  • Changes in ImportType enumeration
    • Video, Midi, ClipGroups values were removed
  • Changes in PaginationRequest message
    • message layout was changed
  • Changes in MemoryLocation message
    • location, track_name, color_index fields were added
  • Changes in MemoryLocationProperties message
    • venue_snapshot_index, venue_snapshot_name fields were added
  • Changes in EditMemoryLocationRequestBody message
    • color_index, location, track_name fields were added
  • Changes in CreateMemoryLocationRequestBody message
    • color_index, location, track_name fields were added
  • Changes in GetMemoryLocationsRequestBody message
    • pagination_request field was added
  • Changes in GetMemoryLocationsResponseBody message
    • pagination_response field was added
