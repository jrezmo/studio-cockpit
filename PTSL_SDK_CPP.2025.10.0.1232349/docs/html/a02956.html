TrackControlBreakpoint

Description

Structure that marks the time and value of a control.

Used for both setting static values and for setting a value at a specific time for automation.

Since
Pro Tools 2025.10

Member Data Documentation

◆ time

TimelineLocation time = 1

Describes where on the timeline an automation breakpoint exists.

A single breakpoint at time 0 is a static value for the control.

◆ value

float value = 2

Establishes the value of an automation breakpoint.

Allowed values depend on the control type for this breakpoint.

Control type control_type value
