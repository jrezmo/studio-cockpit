\hypertarget{a04105_autotoc_md31}{}\doxysubsection{Overview}\label{a04105_autotoc_md31}
The Pro Tools Event System, introduced in the 2025.\+06 release, enables Pro Tools to send real-\/time notifications to subscribed clients. This system allows clients to subscribe to specific events, unsubscribe from them, and receive event notifications as they occur in real-\/time.

The event system consists of three main API calls\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{a00494_aaf8d2069e2c6bbecf4aa8f9819028e8aa3123650ddf7f82ade0c8def87b1f7f40}{Subscribe\+To\+Events}} -\/ Subscribe to specific events
\item \mbox{\hyperlink{a00494_aaf8d2069e2c6bbecf4aa8f9819028e8aabc3d3419d76198a03985575dff948bce}{Poll\+Events}} -\/ Receive real-\/time event notifications
\item \mbox{\hyperlink{a00494_aaf8d2069e2c6bbecf4aa8f9819028e8aa4890bca2c334308b21af02d39e25b1ee}{Unsubscribe\+From\+Events}} -\/ Unsubscribe from events
\end{DoxyItemize}\hypertarget{a04105_autotoc_md32}{}\doxysubsection{Key Concepts}\label{a04105_autotoc_md32}
\hypertarget{a04105_autotoc_md33}{}\doxysubsubsection{Event Subscriptions}\label{a04105_autotoc_md33}
Each subscription is uniquely identified by the combination of\+:
\begin{DoxyItemize}
\item PTSL session
\item Event\+Id
\item Filter data (optional)
\end{DoxyItemize}

Any difference in filter data results in a new subscription, which is managed independently, even if they share the same Event\+Id. For example, you can subscribe to track record-\/enabled state changes for multiple specific tracks by creating separate subscriptions with different track IDs in the filter data.\hypertarget{a04105_autotoc_md34}{}\doxysubsubsection{Event Priority}\label{a04105_autotoc_md34}
All event-\/related APIs are processed with higher execution priority compared to normal API commands, ensuring timely delivery of events without blocking other API operations.\hypertarget{a04105_autotoc_md35}{}\doxysubsection{Workflow}\label{a04105_autotoc_md35}

\hypertarget{a04105_autotoc_md37}{}\doxysubsubsection{Step 2\+: Start Polling for Events}\label{a04105_autotoc_md37}
Use \mbox{\hyperlink{a00494_aaf8d2069e2c6bbecf4aa8f9819028e8aabc3d3419d76198a03985575dff948bce}{Poll\+Events}} to begin receiving real-\/time events. This is a long-\/polling API that streams events using the Intermediate Responses mechanism.

Important Notes\+:
\begin{DoxyItemize}
\item Operates indefinitely without a typical end condition
\item Only one active Poll\+Events connection is allowed per PTSL session
\item If a new Poll\+Events call is made, the previous call is terminated
\item Runs immediately, bypassing the command queue
\item Not affected by the timeout mechanism
\item Does not suppress Pro Tools UI alerts
\end{DoxyItemize}
