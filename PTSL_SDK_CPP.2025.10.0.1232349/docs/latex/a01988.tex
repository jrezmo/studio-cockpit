\doxysubsection{Description}
Parses command line arguments into internal structure of parameters and arguments. 

Following terms are used in the context of this class\+:
\begin{DoxyItemize}
\item \textquotesingle{}parameter\textquotesingle{} (\textquotesingle{}param\textquotesingle{}) is a value that starts with a param\+Symbol (e.\+g., \char`\"{}-\/\char`\"{}) – e.\+g., \char`\"{}-\/file\char`\"{};
\item \textquotesingle{}argument\textquotesingle{} (\textquotesingle{}arg\textquotesingle{}) is a value that follows a parameter – e.\+g., \char`\"{}script.\+txt\char`\"{}. Parameters can have single, multiple or no arguments.
