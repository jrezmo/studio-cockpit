PTSL SDK 2023.\+12 uses \mbox{\hyperlink{a00446_a83c5961efacc6c06e3e75877c239ebb1}{PTSL\+\_\+\+VERSION}} 2 of the client interface

Pro Tools 2023.\+12 is compatible with clients using \mbox{\hyperlink{a00446_a83c5961efacc6c06e3e75877c239ebb1}{PTSL\+\_\+\+VERSION}} 1 and higher

DEVELOPER CERTIFICATES ARE NO LONGER SUPPORTED

You no longer need to obtain a developer certificate to establish a PTSL connection to Pro Tools.
