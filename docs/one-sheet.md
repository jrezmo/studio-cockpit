# Studio Cockpit

Mission control for modern audio production.

## The Problem
Studio operations are fragmented: ingesting files, launching sessions, printing stems, managing settings, and tracking clients and projects live in separate tools and workflows.

## The Solution
Studio Cockpit brings those workflows together in one desktop control surface, including studio business management for clients, projects, tasks, and correspondence.

## Key Benefits
- Faster, cleaner file ingestion
- Consistent session launch flows
- Standardized stem output
- Centralized studio configuration
- Clear visibility into clients, projects, and open tasks
- Session logbook for billable history and follow-ups
- Live Pro Tools integration

## Core Features
- Unified multi-panel dashboard
- Smart Ingest with rules and history
- Client and session management with projects, tasks, and correspondence
- Mixing workflow prep checklist with notes
- Guided session launch
- Stem filename engine
- Pro Tools MCP integration
- Session Stats for archive search and plugin insights

## Best For
- Professional studios
- Post-production teams
- Engineers managing high session volume
